# 分布式方案

#### 概述

* 路由器节点
* 中心节点组集群
* 国家节点组集群
* 区域节点组集群
* 逻辑节点组集群
* 数据库方案

#### 详细方案

* 路由器节点
  * 提供路由服务 
  * 路由通过逐级向上的方法, 
* 中心节点组集群
  * 用于国家之间资源调度, 国家节点的服务器注册与发现
* 国家节点组集群
  * 用于区域内的资源调度，区域节点组的服务器注册与发现， 国家内所有服务器的注册
* 区域节点组集群
  * 用于该区域内的资源调度，区域内的逻辑节点的服务器注册与发现
* 逻辑节点组集群
  * 业务的基本单位，需要注册到区域节点服上或者其他提供路由的服务器组
* 数据库方案
  * 按照功能划分
