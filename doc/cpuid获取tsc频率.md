**CPUID 0x15 的标准返回值：**

| 寄存器 | 作用                     | 举例（2592 MHz） |
| ------ | ------------------------ | ---------------- |
| EAX    | **TSC频率（单位：MHz）** | 2592             |
| EBX    | TSC频率高32位（通常=0）  | 0                |
| ECX    | 0（固定）                | 0                |
| EDX    | 0（固定）                | 0                |

**所以正确代码应该是：**

uint32_t eax, ebx, ecx, edx; __cpuid(0x15, &eax, &ebx, &ecx, &edx); // 直接返回EAX！不需要乘除 return eax; // ✅ TSC频率（单位：MHz）
