* Ai大模型概念

  核心特点：

  * | 特点           | 说明                                             |
    | :------------- | :----------------------------------------------- |
    | **参数规模大** | 通常数十亿到数千亿参数，如GPT-4约1.7万亿参数     |
    | **泛化能力强** | 一个模型能处理多种任务，无需针对每任务单独训练   |
    | **涌现能力**   | 参数规模突破临界点后，出现推理、思考等"涌现"能力 |
    | **多模态能力** | 模型支持理解文字、图片、音频、视频等多模态内容   |
